# RBAC System

## Project Description
Role Based Access Control (RBAC) system using React frontend and Spring Boot backend.

## Tech Stack
Frontend:
- React
- Vite
- Axios

Backend:
- Spring Boot
- JPA
- MySQL

## Features
- User Login
- Role based access
- Employee Management

## How to Run

Frontend:
cd frontend
npm install
npm run dev

Backend:
cd backend
mvn spring-boot:run
